## ūüéģ About the Game

**Jungle Warfair** is a tactical combat game set in dense, procedurally generated jungle environments. Command your squad through hostile terrain, manage resources, and outmaneuver enemies hidden beneath a canopy that hides as many dangers as your opponents do.

- ūüĆŅ **Dynamic jungle environments** ‚ÄĒ terrain, weather, and wildlife all affect combat
- ūüß† **Tactical squad AI** ‚ÄĒ give orders and let your team execute
- ūüĒę **Deep weapon & gear systems** ‚ÄĒ every loadout decision matters
- ūüóļÔłŹ **Procedural map generation** ‚ÄĒ no two missions are the same

## ūüöÄ Installation

### Fresh Install

1. Go to [**Releases**](../../releases) and download the latest stable build for your OS
2. Run the installer (Windows) or extract the archive (Linux/macOS)
3. Launch `JungleWarfair` ‚ÄĒ no additional setup required

### Installing Mods

1. Download a mod (`.zip` or folder)
2. Extract it into your mods directory:
   - **Windows:** `%APPDATA%\JungleWarfair\mods\`
   - **Linux:** `~/.config/JungleWarfair/mods/`
   - **macOS:** `~/Library/Application Support/JungleWarfair/mods/`
3. Launch the game and enable the mod in **Settings ‚Üí Mods**
4. Restart if prompted

### Updating

The game will notify you in-app when a new version is available. You can also watch this repo (ūüĎĀÔłŹ **Watch ‚Üí Releases only**) to get notified on GitHub.

> ‚ö†ÔłŹ **Before updating:** Back up your saves and check if your mods are compatible with the new version.
